保存和恢复
----------

``tf.train.Saver`` 类提供了保存和恢复模型的方法。通过
``tf.save_model.simple_model``
函数可轻松地保存适合投入使用的模型。Estimator 会自动保存和回复
``model_dir`` 中的变量。
